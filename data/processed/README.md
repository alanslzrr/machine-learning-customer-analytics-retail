# Datos Procesados

## Descripción
Esta carpeta contiene los datos después del preprocesamiento y limpieza.

## Estructura Esperada
- `cleaned_data.csv`: Dataset limpio sin valores faltantes
- `scaled_data.csv`: Datos normalizados/escalados
- `encoded_data.csv`: Datos con variables categóricas codificadas
- `feature_selected/`: Datasets con selección de features

## Archivos por Línea de Trabajo
- `clustering/`: Datos preparados para clustering
- `classification/`: Datos preparados para clasificación
- `regression/`: Datos preparados para regresión

## Notas
- Estos archivos se generan automáticamente durante el preprocesamiento
- Cada línea de trabajo puede requerir diferentes transformaciones
