# Datos Originales (Raw Data)

## Archivos Originales
- `proy_supermercado_dev.csv`: Dataset principal del proyecto

## Descripción del Dataset
El dataset contiene información anonimizada de clientes del supermercado incluyendo:
- Variables demográficas
- Comportamiento de compra
- Interacción con campañas de marketing
- Variables de respuesta

## Importante
- Los datos están anonimizados por motivos de privacidad
- No modificar estos archivos directamente
- Siempre trabajar con copias en las carpetas processed/ o interim/
