# Carpeta de Datos

## Descripción
Esta carpeta contiene todos los archivos de datos utilizados en el proyecto.

## Estructura
- `raw/`: Datos originales sin procesar
- `processed/`: Datos después del preprocesamiento
- `interim/`: Datos intermedios durante el procesamiento

## Archivos
- `proy_supermercado_dev.csv`: Dataset original del supermercado
- `train_test_splits/`: Particiones de entrenamiento y prueba
- `feature_engineering/`: Datasets con features creados
