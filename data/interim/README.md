# Datos Intermedios

## Descripción
Esta carpeta contiene datos en estados intermedios del procesamiento.

## Propósito
- Almacenar resultados de pasos intermedios del preprocesamiento
- Facilitar la reproducción del pipeline de datos
- Permitir análisis en puntos específicos del proceso

## Archivos Esperados
- `missing_values_analysis.csv`: Análisis de valores faltantes
- `outlier_analysis.csv`: Análisis de outliers
- `correlation_matrix.csv`: Matriz de correlaciones
- `feature_importance/`: Análisis de importancia de features
