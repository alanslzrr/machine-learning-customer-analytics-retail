# Código Fuente (Source Code)

## Descripción
Esta carpeta contiene todo el código fuente organizado en módulos reutilizables.

## Estructura

### data/
- `load_data.py`: Funciones para cargar datos
- `preprocessing.py`: Funciones de preprocesamiento
- `feature_engineering.py`: Creación de features
- `validation.py`: Validación de datos

### models/
- `clustering_models.py`: Modelos de clustering
- `classification_models.py`: Modelos de clasificación
- `regression_models.py`: Modelos de regresión
- `model_utils.py`: Utilidades para modelos

### evaluation/
- `metrics.py`: Métricas de evaluación
- `visualization.py`: Funciones de visualización
- `cross_validation.py`: Validación cruzada
- `hyperparameter_tuning.py`: Optimización de hiperparámetros

### utils/
- `config.py`: Configuraciones del proyecto
- `helpers.py`: Funciones auxiliares
- `logger.py`: Sistema de logging
- `file_utils.py`: Utilidades para manejo de archivos

## Convenciones
- Cada archivo debe tener docstrings completos
- Funciones modulares y reutilizables
- Manejo de errores apropiado
- Código limpio y bien documentado
