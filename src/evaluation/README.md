# Módulo de Evaluación

## Archivos
- `metrics.py`: Métricas de evaluación
- `visualization.py`: Visualizaciones de resultados
- `cross_validation.py`: Validación cruzada
- `hyperparameter_tuning.py`: Optimización de hiperparámetros

## Funcionalidades

### metrics.py
- Métricas de clasificación: accuracy, precision, recall, F1-score, ROC-AUC
- Métricas de regresión: MAE, MSE, RMSE, R²
- Métricas de clustering: silhouette score, inertia, calinski-harabasz

### visualization.py
- Gráficos de rendimiento de modelos
- Matrices de confusión
- Curvas ROC y precision-recall
- Visualizaciones de clustering
- Gráficos de residuos para regresión

### cross_validation.py
- K-fold cross validation
- Stratified cross validation
- Time series cross validation
- Leave-one-out validation

### hyperparameter_tuning.py
- Grid search
- Random search
- Bayesian optimization
- Early stopping
