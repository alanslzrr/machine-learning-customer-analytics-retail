# Resultados y Métricas

## Descripción
Esta carpeta contiene todos los resultados, métricas y visualizaciones generadas.

## Estructura

### clustering/
- `kmeans_results.csv`: Resultados del clustering K-Means
- `silhouette_scores.json`: Puntuaciones de silhouette
- `cluster_analysis.json`: Análisis de clusters
- `clustering_visualizations/`: Gráficos de clustering

### classification/
- `performance_metrics.csv`: Métricas de rendimiento
- `confusion_matrices/`: Matrices de confusión
- `roc_curves/`: Curvas ROC
- `feature_importance.json`: Importancia de features

### regression/
- `regression_metrics.csv`: Métricas de regresión
- `residual_plots/`: Gráficos de residuos
- `prediction_plots/`: Gráficos de predicción vs real
- `coefficient_analysis.json`: Análisis de coeficientes

### comparison/
- `model_comparison.csv`: Comparación entre modelos
- `cross_validation_results.json`: Resultados de validación cruzada
- `hyperparameter_tuning_results.json`: Resultados de optimización

### visualizations/
- `eda_plots/`: Gráficos del análisis exploratorio
- `model_performance/`: Gráficos de rendimiento
- `business_insights/`: Visualizaciones de insights de negocio

## Formato de Archivos
- CSV para datos tabulares
- JSON para metadatos y configuraciones
- PNG/PDF para visualizaciones
- Estructura consistente para facilitar análisis
